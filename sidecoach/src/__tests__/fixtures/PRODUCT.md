# Project Strategy

## Register
**Product design** - emphasizing user experience and functionality

## Primary Users
Product designers and developers building web applications

## Brand Personality
Modern, clear, and focused on usability

## Anti-References
Designs and patterns to avoid:
- Skeuomorphism
- Over-animated transitions
- Unclear affordances

## Strategic Principles
- Clarity > Decoration
- Accessibility first
- Motion serves meaning
- Constraints enable creativity
